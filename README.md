# MindmapMe
MindMapMe is a modern web-based mind mapping tool that enables users to visually organize ideas using draggable nodes on a dynamic canvas. The application includes a clean authentication interface, real-time editing, export capability, and an AI-style chat assistant for enhanced user interaction.

## Features

- Interactive drag-and-drop nodes  
- Editable mind map elements  
- Export mind map as PNG image  
- Login and signup user interface  
- Clean glassmorphism-inspired design  
- Built-in assistant chat widget  
- Real-time date and time display  
- Responsive layout  
- Modern user experience  

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- html2canvas library  
- Font Awesome Icons
